SignEd Introduction

  • The SignEd tool was developed to simplify the task of creating and managing signed web pages.

  • The tool also automates a number of editing constraints, such as:

    • Preventing overlapping links.

    • Enforcing minimum link timing parameters.

    • Ensuring that the text version of a page does not include links to URL's not in the signed version.
