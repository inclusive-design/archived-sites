Signed Web Page

Long Description: The basic signing web page features are surrounded by a series of colourful graphical icons that help the user navigate the site. The background colour is blue.

The site wraps an icon-based navigation bar and "breadcrumb" mechanism around the basic signed Web page objects. An "English" option turns on the optional text and labels.
