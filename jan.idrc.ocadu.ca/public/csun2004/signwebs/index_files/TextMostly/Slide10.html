Inter-Page Navigation III

  • In a signed video, linguisitic concepts are temporal (located in time) and sequenced.

    • This makes ASL content similar in some ways to speech output.

  • The sequential display of ASL rules out unassisted whole-document views.

    • If there is to be single action linking and an overall link gestalt it will have to be explicitly provided.
