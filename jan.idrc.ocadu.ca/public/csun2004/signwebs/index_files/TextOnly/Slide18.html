Accessibility

  • Surfing the signing webs of the future, users will have varying accessibility requirements.

  • The following accessibility features are built in:

    • Optional text content and text labels provide a text equivalent that can be accessed via Braille displays or Text-to-Speech.

    • Link buttons, text links, and any form controls in the optional text area are keyboard accessible.

    • An interpreted voice-over may be added to videos.

    • Captioning may be added to videos.
